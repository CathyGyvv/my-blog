
函数柯里化的作用
参数复用
提前返回 – 返回接受余下的参数且返回结果的新函数
延迟执行 – 返回新函数，等待执行

```js
const curry = (fn,...args)=> args.length<fn.length?(...arguments)=>curry(fn,...args,...arguments):fn(...args);


function sumFn(a,b,c){
  return a+b+c
}


var sum = curry(sumFn);
console.log(sum(2)(3)(5));
console.log(sum(2,3,5));
console.log(sum(2,3)(5));
console.log(sum(2)(3,5));
```
