对象到字符串的转换：
  如果对象有toString方法，调用，如果它返回一个原始值，js将这个值转换为字符串，返回结果
  如果对象没有toString方法，或者这个方法不返回一个原始值，那么调用valueOf方法，如果存在valueOf，调用它，如果返回值是原始值，js将转换为字符串，返回此结果
  无法从这两个方法获得原始值，抛错
