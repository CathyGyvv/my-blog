子组件：
```js
<div 
  ref={el => {
    if (this.props.inputRef) {
      this.props.inputRef(el); // 调用父组件传入的ref回调函数
    }
    this.inputElement = el;
  }}>
    12111
  </div>
```

```js
 componentDidMount() {
      console.log(this.inputElement, 'refffff');
    }

```
