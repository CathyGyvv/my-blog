meta标签：自动刷新跳转 
```js
<meta http-equiv="Refresh" content="5; URL=page2.html"> 
// 5s 后跳转到page2.html 


<meta http-equiv="Refresh" content="60">
// 每隔60s去刷新,缺点：不好修改和取消。
```
