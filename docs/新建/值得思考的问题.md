## 如何让(a == 1 && a == 2 && a == 3) 的值为 true???
  1.方法一
  ```js
  // 部署 [Symbol.toPrimitive] / valueOf/ toString 皆可
  // 一次返回 1，2，3 即可。
  let a = {
      [Symbol.toPrimitive]: (function(hint) {
              let i = 1;
              // 闭包的特性之一：i 不会被回收
              return function() {
                  return i++;
              }
      })()
  }
  ```

  2.方法二
  ```js
  let a = new Proxy({}, {
      i: 1,
      get: function () {
          return () => this.i++;
      }
  });
  ```

  3.数组的 toString 接口默认调用数组的 join 方法，重新 join 方法
  ```js
  let a = [1, 2, 3];
  a.join = a.shift;
  ```
